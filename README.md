# ReserveX

> A scalable high-concurrency reservation platform built using Spring Boot, React, Redis, Kafka, PostgreSQL, and AI-powered services.

## 🚀 About

ReserveX is a backend-focused full-stack project designed to explore real-world distributed system concepts and scalable software architecture.

The project will evolve from a basic reservation system into a production-style platform capable of handling high concurrency, distributed caching, asynchronous messaging, and intelligent recommendations.

## 🛠️ Tech Stack

### Backend
- Java
- Spring Boot

- Spring Data JPA
- PostgreSQL
- Redis
- Apache Kafka

### Frontend
